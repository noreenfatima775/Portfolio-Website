# Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, skills, and contact information.

## 🚀 Features

- Responsive design for all devices
- Smooth scrolling navigation
- Interactive animations
- Projects showcase section
- Skills section
- About me section
- Contact form / contact section
- Clean and modern UI

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & animations
- **JavaScript (ES6)** – Interactivity
